# HCL-Model
Simulating the Circadian Oscillator part of the Homeostatic-Circadian-Light (HCL) Model as published in Skeldon et al., PLoS Computational Biology (2023).  

run_HCL_model.m - code to run the model simulation and plot output

Set_Paramters.m - function to set initial condition values and parameter values 

HCL_Model.m - function with all the model equations
